NSERC Undergraduate Student Research Award

I worked with Professor Tarek Abdelrahman at the University of Toronto, to prototype a pipelined Deep Neural Network on an FPGA platform to demonstrate the viability of hardware acceleration during training. I prototyped the design in Verilog for the DE1-SoC FPGA board. I learned about pipelined hardware design, neural networks and gained a deep understanding of the underlying math involved intraining and inference.

Python Time Tracker

I used Python to create an application that keeps track of what apps I used and how long I used them. I used the matplotlib and tkinter libraries for the GUI design.

C++ Map Application

I worked in a team of three to create a Google Maps styled GIS in C++. The map was built on the OpenStreetMap api and involved designing the GUI and creating the path finding algorithm for a Pickup and Delivery Traveling Salesman Problem. On the technical side, I learned about the C standard library, debugging, basic multithreading, path finding algorithms such as Dijkstra and A-star and version control (git). Equally important, I learned a lot about project management, collaborative programming and communication.

Python Aim Trainer

This past summer I got into Valorant, (a competitive First Person Shooting game) and I created this as a fun project and also to hopefully improve my mouse movement. I created this in Python using the Pygame library. Features include "flick" and tracking training of varying difficulty.

C/Assembly Racing Game

I worked in a team of two to create a racing game in C for the DE1-SoC FPGA board. The game was output through the VGA display and used a PS2 keyboard and the hardware keys for controls.

Verilog Simon Game

I worked in a team of two to recreate the game Simon on the DE1-SoC FPGA board. The game was output through the VGA display and used a PS2 keyboard as controls for the cursor. The game included randomized levels of increasing difficulty.

NewHacks 2019

I worked in a team of four to create a plant water management program in 24 hours. The program used Arduino sensors to measure the moisture levels in the soil and in Python, a gui notifying the user how much water to give the plant. The Python program also sent a text message to the user, notifying them when the plant needed watering. Our team placed 1st in the sustainability category.
